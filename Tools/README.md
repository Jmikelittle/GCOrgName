Tools

This folder is for a variety of tools which help improve data quality.

**Note:** The `website_validator.py` script is currently not functioning.

### Summary of Scripts

- `data_cleaner.py`: Cleans and preprocesses raw data files, removing duplicates and correcting formatting issues. Produces cleaned data files ready for analysis (e.g., `cleaned_data.csv`).
- `report_generator.py`: Generates summary reports from the cleaned data, including statistics and visualizations. Produces PDF and HTML reports (e.g., `summary_report.pdf`, `summary_report.html`).
- `data_analyzer.py`: Performs advanced data analysis, including trend analysis and predictive modeling. Produces analysis results in CSV and JSON formats (e.g., `analysis_results.csv`, `analysis_results.json`).
- `file_converter.py`: Converts data files between different formats (e.g., CSV to JSON, Excel to CSV). Produces converted data files in the specified format (e.g., `converted_data.json`, `converted_data.csv`).
